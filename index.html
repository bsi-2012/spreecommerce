O que e?

O que e?


Spree Commerce e um e-commerce open-source criado por Sean Schofield em 2007.

Licenca

Licenca


O Spree Commerce opera sobre a licenca BSD (Berkeley Software Distribution) que pode ser modificado sem nenhuma restricao. Esta licenca e similar a GPL.

Tecnologias

Tecnologias

  • Uma solucao baseada em Ruby
  • Utiliza um framework open-source de aplicacoes web chamado Rails
  • Pode operar com qualquer banco de dados suportado pelo framework
