O que é?

O que e?


Spree Commerce é um e-commerce open-source criado por Sean Schofield em 2007.

Licença

Licença


O Spree Commerce opera sobre a licença BSD (Berkeley Software Distribution) que pode ser modificado sem nenhuma restrição. Esta licenca é similar a GPL.

Tecnologias

Tecnologias

  • Uma solução baseada em Ruby
  • Utiliza um framework open-source de aplicações web chamado Rails
  • Pode operar com qualquer banco de dados suportado pelo framework
